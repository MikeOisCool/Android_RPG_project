# Android RPG Project

Ein Android-RPG, entwickelt mit Kotlin und Jetpack Compose.

## Aktuelle Features

- Kampfsystem
- Kritische Treffer
- Ausweichen
- XP-System
- Levelsystem
- Gegner-Skalierung
- Inventarsystem
- Heiltränke
- Heiltrank-Drops
- Inventar-Screen
- Navigation
- Hoch- und Querformat

## Geplante Features

- Großer Heiltrank
- Mehrere Itemtypen
- Waffen
- Ausrüstung
- Händler
- Quests
- Speichern und Laden

## Technologien

- Kotlin
- Jetpack Compose
- ViewModel
- StateFlow
- Navigation Compose
